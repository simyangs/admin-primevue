UI 적용 팁

  • • 버튼의 최소 너비는 4글자 기준(약 63px)으로 설정되어 일관성을 유지합니다.
  • • 조회와 초기화 버튼은 검색 영역 우측 하단 또는 그리드 마지막 열에 배치합니다.
  • • 엑셀 다운로드는 보통 데이터 테이블의 툴바 좌측 또는 우측 상단에 위치합니다.
  • • 처리 버튼(확정, 저장)은 페이지 최하단 액션 바에 주로 배치합니다.
